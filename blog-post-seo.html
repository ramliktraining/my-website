🚀 How I Built My Personal Website

Published on June 2025 by Ramlik Mohamed

I started with a blank HTML file and slowly built out features like:

  • 💻 Responsive layout with flexbox and media queries
  • 🎨 Custom hero banners and parallax effects
  • ✨ Scroll animations using AOS
  • 📦 PWA support with service workers
  • 📈 GA4 Analytics and social meta tags
  • 🔗 Hosted on GitHub Pages (100% free)

Highlights

My favorite part was implementing the typewriter effect and making the whole site feel like a native app. The counter animations also added a nice touch!
