📈 10 SEO Tricks I Used for My Portfolio Site

Published: June 2025 · By Ramlik

1. Descriptive Meta Tags

I made sure every page has a clear and keyword-rich title and description meta tag.

2. Open Graph + Twitter Cards

These help social platforms show rich previews when my links are shared.

3. Clean and Semantic HTML5

Using <main>, <section>, <article>, and headings in proper order helped crawlers understand my layout.

4. Alt Tags on Images

All images include meaningful `alt` text to improve accessibility and indexability.

5. Fast Load Times

I optimized my images, used preconnect for fonts, and served assets via CDN when possible.

6. Mobile-First Design

Google prioritizes mobile usability — so I ensured full responsive layouts.

7. Page Speed Optimizations

Minimized CSS and JS, deferred scripts, and enabled PWA caching.

8. Structured Content

Used logical heading hierarchy (H1 > H2 > H3) and kept text content readable.

9. Robots Meta Tag

Set `` to ensure crawlers index everything.

10. Hosting with HTTPS

My GitHub Pages site is served over HTTPS for security and SEO boost.

Takeaway: You don’t need plugins — just basic good practices make a huge difference.
