Why I Chose Barba.js

To create seamless transitions between pages, reduce reloads, and keep animations flowing smoothly.

Setup Process

I loaded Barba via CDN, added `data-barba` attributes to each page, and wrote JS to define transitions.

Fade Between Pages

Using `leave` and `enter` hooks in Barba, I created a fade-out + fade-in effect on every link click.

JS Example:


barba.init({
  transitions: [{
    leave({ current }) {
      return gsap.to(current.container, { opacity: 0 });
    },
    enter({ next }) {
      return gsap.from(next.container, { opacity: 0 });
    }
  }]
});

What Improved

It gave my site a slick app-like feel. Pages feel faster, users stay longer, and the flow is delightful.

Bonus: Re-initialize AOS


barba.hooks.afterEnter(() => {
  AOS.init({ once: true });
});
