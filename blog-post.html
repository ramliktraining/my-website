How I Built My PWA Portfolio

By Ramlik Mohamed • June 18, 2025

1. Planning the Layout

I began with a basic wireframe of my website’s structure: Home, About, Services, Portfolio, Contact. This helped shape content and layout priorities.

2. Making It Responsive

Using media queries and flexible grid layouts, I ensured my site looked great across all screen sizes, from mobile to desktop.

3. Adding Modern Features

  • ✅ Scroll animations with AOS
  • ✅ Parallax hero with fade-ins
  • ✅ Animated counters and typewriter effects
  • ✅ SPA-like transitions using barba.js
  • ✅ PWA manifest and service worker for installable app

4. Hosting & Analytics

I deployed my site on GitHub Pages, connected Google Analytics (GA4), and added social sharing meta tags for SEO visibility.
